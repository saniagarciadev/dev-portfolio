Project 01

‘Port’

A minimal, yet versatile, communication system.

Project details

'Port' | Web app | First published: December 2020

Github repo

Goal

To create a chat app that's uses cutting edge technology to feel more organic, intimate and tactile than the ones we commonly use. With some unique, fun features (such as live typing), no account required to start, fluid animations and the darkest dark mode ever.

How I accomplished it

First I built a prototype where messages where sent through fetch requests that would store and retrieve message/user data from a database [React, Node/Express, REST API, MongoDB/Mongoose]. Then I created an early visual design system [Figma] and a features roadmap, and proceeded to architect the instant messaging system, while I studied and prototyped with websockets [Socket.io]. I completed a minimum viable product and took it live [Netflify, Heroku].

Project 02

Tana's art gallery

Letting art speak for itself.

Project details

'Tana's art gallery' | Artist showcase | First published: January 2021

Visit siteGithub repo

Goal

To create the most essential and clean, but also professional-looking artist showcase I could make. To give the artist pride in her craft and trajectory, being that she's been an enthusiast artist for many years without much public recognition.

How I accomplished it

I started by sketching a visual identity [Figma] and decided on the white theme to evoke a gallery experience. I built the first protope as a static-only site, but moved it to React/Gatsby/Contentful when the artist and I agreed on adding a blog at a later time. I created a visual database for the artist to easily input information on each work [Google Sheets] and set up a system for the site to programatically read this data. I set up a contact form with Netlify Forms and Zapier.

Project 03

Jose's Barbershop

Easy access, happy customers.

Project details

'Jose's Barbershop' | Storefront | Prototype

Goal

To design a modern and attractive storefront for a small business, where customers can easily get relevant information and book an appointment.

How I accomplished it

I did a thorough research on sites of similar businesses and took note of many wins and errors. I designed a visual identity that would express an modern and dynamic attitude, while still showing love for the tradition of the barberhop ritual. I created mockups for different screen sizes [Figma, Photoshop].
